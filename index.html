Rules:

  1. Any live cell with fewer than 2 live neighbours dies, as if caused by underpopulation.
  2. Any live cell with 2 or 3 live neighbours lives on to the next generation.
  3. Any live cell with more than 3 live neighbours dies, as if by overpopulation.
  4. Any dead cell with exactly 3 live neighbours becomes a live cell, as if by reproduction.

Instructions:

  • Click/Tap on the cells in the table to change their state from dead to alive.
  • Click "Run" button to start a automatic run of the cycle.
  • Click "1 cycle" button to run exactly one generation of the game.
  • Click "10 cycles" button to run exactly ten generations of the game.
  • Click "Reset" button to clear the table and information about current run.
